# Retail Sales Forecasting

## Project Overview
This project predicts retail sales using machine learning.
It processes historical sales data and trains models to forecast future sales.

## Features
- Data preprocessing
- Feature engineering
- XGBoost model
- Hyperparameter tuning (Optuna)
- Performance evaluation
- GitHub-ready structure

## Model Performance
- R2 Score: ~0.42
- MAE: 0.93
- RMSE: 1.18

## Technologies
- Python
- Pandas
- Scikit-Learn
- XGBoost
- Optuna

## How to Run
```bash
pip install -r requirements.txt
python src/train.py
